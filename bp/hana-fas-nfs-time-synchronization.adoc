= 時間同步
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
您必須同步儲存控制器與SAP HANA資料庫主機之間的時間。若要這麼做、請為所有儲存控制器和所有SAP HANA主機設定相同的時間伺服器。
