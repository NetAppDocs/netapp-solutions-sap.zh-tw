= 在目標主機上掛載新的磁碟區
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
現在可以根據先前建立的「etc/stb'」檔案、將新的磁碟區掛載到目標主機上。

....
vm-pr1:~ # mount -a
....
